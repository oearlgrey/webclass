overflow

overflow-흘러너친 컨텐츠에 대해서 대응하기위 css 속성이다.

  • hidden - 흘러넘친 컨텐츠에 대해서 안 보이게 처리
  • scroll - 흘러넘친 컨텐츠에 대해서 스크롤바를 표현
  • auto - 흘러넘친 컨텐츠에 대해서 스크롤바를 표현
  • visible - 흘러넘친 컨텐츠에 그대로 보여주도록 처리
