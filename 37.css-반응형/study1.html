반응형 css

화면의 너비에 따라 다른 스타일의 css가 적용되도록 스타일을 입력하는 방법

보통 media-query 라고 부른다

화면너비의 분기점(break point)

  • pc화면 - 보통 화면너비 최소값 1024픽셀
  • tablet화면 - 768픽셀 이상 1024픽셀 미만
  • moblie화면 - 768픽셀 미만

모바일 css화면을 스타일링 할 때는 가로사이즈는 되도록이면 상대단위(%)를 사용한다.

세로사이즈는 되도록이면 auto 를 활용한다.

픽셀 말고 % 사용하기

vw단위는 쓰려면 화면너비에 대한 비율로 나누기 디자인한 너비시안 / 340 전체 화면 너비

% 부모의 태그에 대한 너비의 비율임 ex 로고의 너비를 지정해주시고 싶음 디자인한 너비시안 ex innerheader / 컨텐츠 영역 ex 300 = %

ex 전체 영역 340px 컨텐츠 영역(가이드라인) 303px

% 계산 컨텐츠 영역 / 전체 영역 x 100 = %

303 / 340 x 100 = 89.11754... = 90% 비율 % = 340분의 303 x 100

vw 계산 전체영역 너비 100vw (340px)에 대한 비율

ex 메뉴단 아이콘 너비 22px -> 상대단위로 변경

% 단위로 사용시 - 22 / 303 x 100 = 7.26% vw 단위로 사용이 - 22 / 340 x 100 = 6.4vw .btnMoMenu - > vw를 사용한 이유 %를 사용하면 부모의 높이를 상속하기 때문에 정사각 모양 x % , vh, vw 모두 결과적으로 픽셀로 계산됨 rem과 동일 rem를 써도 무방함 (rem은 폰트에만 적용되는 것은 아님)
